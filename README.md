# Rosario Inventory Pro v1.0 🚀

Sistema de gestión de inventario y ventas desarrollado con **PHP 8.2** y **MySQL**.

### ✨ Características:
- Gestión de stock con alertas de cantidad crítica.
- Registro de ventas con transacciones seguras (PDO).
- Reportes dinámicos exportables a Excel.
- Historial de auditoría para control de usuarios.

### 🛠️ Instalación local:
1. Clonar el repositorio.
2. Importar `sistema_inventario.sql` en XAMPP.
3. Configurar `config/db.php`.
