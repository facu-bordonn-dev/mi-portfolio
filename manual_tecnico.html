SISTEMA DE STOCK Y VENTAS v1.0

Manual de Usuario y Documentación Técnica del Sistema

Este documento detalla el funcionamiento del sistema de gestión desarrollado para Rosario Empresa, optimizado para el control de stock, ventas y auditoría interna.

1. Arquitectura del Sistema

Tecnologías utilizadas:
  • PHP 8.2 Lógica de servidor y procesamiento de datos.
  • MySQL Base de datos relacional para almacenamiento seguro.
  • PDO Conexión segura con sentencias preparadas contra inyección SQL.
  • Bootstrap 5 Interfaz adaptativa y profesional.

2. Módulos Principales

A. Dashboard de Control

Panel central que muestra en tiempo real la salud financiera del inventario:

  • Variedad: Total de productos únicos registrados (SKUs).
  • Valorización: Cálculo automático del valor total de la mercadería en estantería.
  • Alertas de Stock: Notificación visual automática cuando un producto tiene menos de 5 unidades.

B. Gestión de Ventas

Módulo de salida de mercadería con validación de existencia:

  • Descuento automático de stock tras confirmación.
  • Registro en tabla de ventas para historial de ingresos.
  • Bloqueo de ventas si la cantidad solicitada supera el stock físico.

3. Reportes y Auditoría

Tipo de Reporte Descripción Formato
Inventario Total Listado completo de productos y valores. Excel (.xls)
Rango de Fechas Detalle de ventas en un periodo específico. Excel (.xls)
Historial Log Auditoría de quién realizó cada movimiento. Vista Web

4. Seguridad y Roles

El sistema implementa dos niveles de acceso:

  • Administrador: Control total, edición de productos, eliminación y gestión de usuarios.
  • Empleado: Visualización de stock y registro de ventas únicamente.
