BookingSystem v1.0

Manual de Integración y Usuario Final

Especificaciones

  • Backend: PHP 8.2 (PDO)
  • DB: MySQL Relacional
  • Notificación: WhatsApp Link API
  • Auth: Bcrypt Password Hash

Guía de Funcionamiento

Registro de Clientes

El sistema captura el nombre, correo y teléfono del usuario. El número telefónico es validado mediante expresiones regulares para asegurar que el Administrador pueda contactarlo sin errores de formato.

Gestión de la Agenda

Desde el Panel Admin, el encargado visualiza los "Turnos de Hoy" y los "Pendientes". Cada turno tiene un estado (Pendiente, Confirmado, Completado, Cancelado) que permite filtrar la carga de trabajo diaria.

Confirmación y WhatsApp

Al confirmar un turno, se habilita dinámicamente el botón de Notificar. Este genera un enlace URL dinámico que abre WhatsApp Web o la App móvil con un mensaje personalizado: 

Hola [Cliente]! Te confirmo tu turno para [Servicio] el día [Fecha]...

Instalación Local

  1. Importar el archivo database.sql en phpMyAdmin.
  2. Configurar los parámetros de conexión en config/db.php.
  3. Asegurarse de tener habilitada la extensión PDO en el servidor Apache.
  4. Acceder vía localhost/booking-system.
